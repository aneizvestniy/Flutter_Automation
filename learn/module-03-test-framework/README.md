# Module 03 — Test Framework & First E2E Tests (Week 2–3)

## Goal
Set up an E2E framework for Flutter using Appium 2 + `appium-flutter-driver`.

## Week 2 milestone
By the end of week 2, you should have your first E2E test:
- login flow (5–7 steps)

## What we build
- Appium project under `testing/e2e_appium/`
- Page Objects
- Test data management (fixtures)
- Base E2E suites:
  - login
  - CRUD
  - full 20+ step scenario
