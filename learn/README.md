# Learning Track (4 weeks)

This folder contains the week-by-week learning modules.

- `module-01-dart-flutter-basics/` (Week 1)
- `module-02-training-app/` (Week 1–2)
- `module-03-test-framework/` (Week 2–3)
- `module-04-advanced/` (Week 3–4)

Each module includes:
- `README.md` with outcomes, timebox, and checklist
- `exercises/` with hands-on tasks and acceptance criteria
- optional `examples/` with runnable snippets
